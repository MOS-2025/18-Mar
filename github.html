Git Commands for Visual Studio Code

1. Basic Git Commands

CommandDescription
git initInitialize a new Git repository
git clone <repo-url>Clone a repository
git statusShow current branch status
git add .Stage all changes
git commit -m "message"Commit staged changes
git push origin <branch>Push commits to remote
git pull origin <branch>Fetch and merge changes

2. Undo & Reset Commands

CommandDescription
git reset --hard HEADDiscard all local changes
git restore <file>Discard changes in a file
git stashSave uncommitted changes

3. Checking History & Logs

CommandDescription
git log --oneline --graphView commit history graph
git diffShow changes between commits
